# Nomad System

### Esté projeto consiste em uma plataforma de gestão de vendas weblocal, visado e pensado em micro e pequenas empresas, infelizmente o repositorio ainda está privado para garantir a segurança da construção e finalização do projeto que se constitue de uma equipe de 5 pessoas, dividido entre a equipe de desenvolvimento e a equipe de documentação, a idea ainda é bem simples que contem as basicas mas importantes e necessarias funcões para sua finalidade;

#### Optamos por usar a linguagem java com o framework spring boot para o backend do projeto e integrado a esse sistema, decidimos usar a dependencia do thymeleaf para a construção das telas que usam html, css e javascript, para a idealização das telas usamos o figma para pensar em uma ideia mais fluida para os usuarios.

### O sistema nomad conta com dois perfis:

#### Administrador:

##### Que conta com as telas de gerenciamento geral de sua empresa, e as funcionalidades para auxiliar neste perfil;

#### Funcionario:

##### Conta com as funcionalidades de gerenciamento direto, voltado para vendas e cadastro de clientes;


### Ideias

#### Pretendemos aprimorar esté projeto para uma plataforma totalmente online, onde os usuários terão a possibilidade de realizar o download e gerir suas empresas de forma remota com armazenamento dos dados em nuvem;
